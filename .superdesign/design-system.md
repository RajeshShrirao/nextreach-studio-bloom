# Design System

- Brand: NextReach Studio repositioned as an async Docs-as-a-Service studio for indie SaaS founders.
- Tone: crisp, technical, credible, async-first.
- Layout: narrow reading width for copy, strong section dividers, code/docs-inspired cards.
- Visual language: editorial typography, warm paper background, dark ink surfaces, green signal accents.
- Core promises to surface repeatedly: done-for-you docs, live docs site, 5-day delivery, async collaboration, fixed scope packages.
